# Simple Interest Calculator

This project is a simple interest calculator created as part of the IBM Full Stack Software Developer course.

## Description

The Simple Interest Calculator calculates the simple interest based on the following formula:

Simple Interest = (Principal × Rate × Time) / 100

Where:
- **Principal** is the initial amount of money
- **Rate** is the annual interest rate
- **Time** is the time period in years

## Example

If the principal amount is 1000, the rate of interest is 5%, and the time period is 2 years:

Simple Interest = (1000 × 5 × 2) / 100 = 100

## Usage

This calculator can be used to quickly estimate the interest earned on an investment or loan using the simple interest formula.
